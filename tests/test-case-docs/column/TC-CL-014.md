# 测试用例：TC-CL-014

## 测试目的

验证 Column 组件与 Row 组件组合使用时的布局表现，确保嵌套和组合场景下各自的排列方向、间距、对齐等属性能正确生效。

## 测试环境

- GUIDS2HTML 转换器版本：1.0.0 或更高
- 支持的浏览器：Chrome 90+, Firefox 90+, Safari 14+, Edge 90+

## 前提条件

- 已正确安装和配置 GUIDS2HTML 转换器
- 已准备好测试用例 JSON 文件 `tests/data/components/column/TC-CL-014.json`

## 测试步骤

1. 加载测试用例 JSON 文件
2. 使用 GUIDS2HTML 转换器将 JSON 转换为 HTML
3. 在支持的浏览器中打开生成的 HTML 文件
4. 观察 Column 与 Row 组合后的渲染结果

## 测试数据

测试用例 JSON 结构包含：

- Column 组件内嵌套 Row 组件，Row 组件内有多个子元素
- Row 组件内嵌套 Column 组件，Column 组件内有多个子元素
- 多层嵌套和交错组合
- 各层设置不同的间距、对齐方式、背景色等

## 预期结果

1. Column 组件内的 Row 组件应水平排列其子元素，Column 组件自身应垂直排列
2. Row 组件内的 Column 组件应垂直排列其子元素，Row 组件自身应水平排列
3. 各层的间距、对齐、背景色等样式应分别正确生效
4. 组合嵌套结构应清晰、无错位或溢出

## 验证点

- 验证 Column 与 Row 组合时的排列方向和嵌套效果
- 验证多层嵌套下的样式独立性
- 验证间距、对齐等属性在组合场景下的表现
