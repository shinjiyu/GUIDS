## 为什么我们重组了文档？

随着 GUIDS2HTML 功能的不断扩展，单个文档文件变得过于庞大且难以维护。新的文档结构提供以下好处：

1. **模块化** - 每个组件和系统有专门的文档
2. **可扩展性** - 更容易添加新组件的文档
3. **易于导航** - 清晰的目录结构
4. **并行更新** - 多人可同时更新不同部分
5. **版本控制** - 更细粒度的变更跟踪
